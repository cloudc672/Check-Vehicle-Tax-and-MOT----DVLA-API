# Check-Vehicle-s-Tax-and-MOT----DVLA-API
As a legal requirement vehicles must have valid tax and MOT to be driven on public roads. This GUI-based Python application allows users to enter a vehicle registration number and retrieve its MOT and tax status using an official UK DVLA API.

Instructions to Set Up 
1. Gain an API key from DVLA by requesting for one.
2. Clone Repository from Git
3. Re-Launch VS-Code as administrator Download python from python.org use default settings restart computer Relaunch VSCode and Powershell as administirator
4. Check whether python and pip is installed on your device:
    python --version
    pip -- version
   
5. Install these packages in order:
    pip install customtkinter
    pip install subprocess

6. Add your API key in file called validator.py under API_KEY = "", where your secret API key would go in-between the quotes.
7. Run the program "python GUI.py"
